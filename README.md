# Gestionnaire de Bases de Données

Une application de bureau (WPF / .NET) conçue pour se connecter et gérer de multiples types de bases de données (PostgreSQL, MySQL, MongoDB) depuis une seule et unique interface moderne et unifiée.

## Fonctionnalités

- **Multibase :** Prise en charge native de PostgreSQL, MySQL et MongoDB.
- **Gestion des Profils :** Sauvegarde et gestion de plusieurs profils de connexion.
- **Explorateur de Schéma :** Navigation intuitive à travers les bases de données et leurs tables.
- **Opérations CRUD basiques :** Création et suppression de bases de données et de tables depuis l'interface graphique.
- **Éditeur de Requêtes :** Exécution de requêtes personnalisées avec un tableau de résultats dynamique.

## Prérequis

- [.NET 8.0 SDK](https://dotnet.microsoft.com/download/dotnet/8.0) (ou version ultérieure)
- Système d'exploitation Windows (car l'application utilise l'interface WPF)

## Lancement du Projet

1. Clonez ce dépôt sur votre machine.
2. Ouvrez le projet dans votre IDE favori (Visual Studio 2022, Rider, ou VS Code).
3. Restaurez les packages NuGet (les dépendances comme Npgsql, MySql.Data et MongoDB.Driver).
4. Compilez et lancez l'application.
